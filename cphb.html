Cphbusiness

I am currently studying at Cphbusiness, where I study Multimedia design. This education is divided into four semesters. Every semester contains four so called "flows", and every flow has a different theme.

Semesters

01

Digital image processing and website prototype Adobe Photoshop & Figma

I learned how to manipulate picture in Adobe Photoshop by removing andadding elements, adjusting the brightness or giving the image filters,and how to adjust the image size with crop, orientation and format size

In Figma I learned how to make a website prototype. How to style andsize each page, how to go from page to another and how to turn items intocomponents.

02

Visual concept for digital and printed media Adobe Illustrator & Figma

I learned how to draw and create elements in Adobe Illustrator. How to trace an image, and genrally using the different tools available.

In Figma I learned how to make a landingpage and how to add animations. Also how to make scoll buttons.

03

Build your first website HTML & CSS

I learned basic HTML and CSS, and how to build a website. Also how to make a website suitable to different size screens, like desctop and mobile, and how to always do "mobile first".
